# Инкрементальная загрузка данных об объявлениях на автомобили

Это приложение предназначено для инкрементальной загрузки данных об объявлениях на автомобили. Оно позволяет загружать и обновлять информацию о доступных автомобилях, сохраняя при этом данные, которые уже были загружены.

## Установка

### Предварительные требования

- Python 3.6 или выше
- Библиотеки:
  - pandas
  - requests (или другая библиотека для работы с API)

### Установка зависимостей

Для установки необходимых библиотек используйте pip:

```bash
pip install pandas requests
```

## Использование

1. **Конфигурация**: Отредактируйте файл конфигурации, чтобы задать параметры подключения к API или базе данных.

2. **Запуск приложения**: Для запуска приложения выполните следующий команду:

```bash
python main.py
```

3. **Инкрементальная загрузка**: Приложение автоматически сопоставляет новые данные с уже существующими записанными автомобилями и обновляет их по мере необходимости.

## Примеры

Вы можете использовать приложение для получения данных о различных моделях автомобилей следующим образом:

```python
from car_data_loader import CarDataLoader

loader = CarDataLoader(api_key='YOUR_API_KEY')
loader.load_data()
```

## Функциональность

- Загружает новые объявления об автомобилях.
- Обновляет существующие записи.
- Удаляет устаревшие данные (по запросу).
- Хранит историю изменений данных для дальнейшего анализа.
